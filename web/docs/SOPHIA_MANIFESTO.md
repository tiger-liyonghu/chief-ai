# Sophia 纲领

**你在哪，就安心在哪。一个都不丢。**

Sophia — 中小企业和超级个体的 AI 幕僚。

---

## Sophia 是什么

你只能在一个地方。你的客户、工作、家庭分布在所有地方。

Sophia 帮你盯着你不在的地方。该静就静，该响就响。

她从你的邮件、WhatsApp、日历里，自动知道你答应了谁什么、谁在等你、什么快到期了。不需要你录入任何东西。

---

## 三个方向

### 🤝 客户不丢 — 收入

你答应客户的每一件事，Sophia 都记着。

- 你答应 Lisa 周五给报价 → Sophia 盯着，逾期了提醒你
- Kevin 一个月没联系了 → Sophia 告诉你关系在冷却
- 客户邮件里提到"我朋友也需要" → Sophia 标记为转介绍机会
- 保单 45 天后到期 → Sophia 提醒你该联系续保了
- 要见客户了 → Sophia 准备好这个人的背景、历史、未了承诺

### 🧳 差旅不乱 — 效率

出差是你最忙、最容易丢球的时候。Sophia 在这个场景下最强。

- 你飞吉隆坡 → Sophia 告诉你 KL 有 3 个客户可以顺便见
- 到了新城市 → 落地简报：待办、客户、文化注意事项
- 见客户前 30 分钟 → 背景、上次聊了什么、未了承诺，全准备好
- 出差回来 → 这趟出差的所有跟进事项，列好了
- 你在东京，客户在新加坡 → Sophia 帮你算好最佳联系时间

### 🏠 家庭不忘 — 底线

家庭时间是硬约束。Sophia 默认站在家庭这边。

- 客户约了周六下午 → Sophia 说"这个时间是 Emily 钢琴课"
- 你出差在外 → Sophia 记着你对家人的承诺（"爸爸周六回来"）
- 老婆生日、结婚纪念日、孩子考试 → Sophia 提前提醒
- 工作要占家庭时间 → 只有你主动调整，Sophia 不会自动让步

**客户是收入，差旅是杠杆，家庭是底线。**

---

## 三个动作

不管哪个方向，Sophia 做的事就三个：

**盯着。** 你在忙别的，Sophia 看着你所有的承诺、客户、日程。大部分时间她不说话。

**提醒。** 往后看：Lisa 的报价逾期了、Kevin 一个月没联系了。往前看：今天下午 3 个背靠背客户会议、周三和女儿接送冲突了。

**准备。** 你要见客户了，Sophia 提前把这个人的背景、历史、未了承诺准备好。你到就能用。

## 谁用 Sophia

管很多客户、经常出差、家里有家人要顾的人。

保险代理人、独立顾问、律师、小老板。一个人当老板又当销售又当爸爸/妈妈的那种。

保险代理人是第一批用户——他们是最典型的超级个体：大量客户关系、频繁出差、承诺密集、家庭要顾。

## 5 分钟见效

连上 Gmail。Sophia 扫描 90 天邮件。5 分钟后告诉你：

- 你忘了回复谁
- 你答应了谁什么还没做
- 哪些重要的人很久没联系了
- 下周有什么冲突

你看到的第一反应应该是："操，这个我确实忘了。"

## 六条规矩

1. **不撒谎。** 不知道就说不知道。
2. **不自作主张。** 所有对外动作你确认才发。
3. **不该烦的时候不烦。** 沉默是美德。
4. **家庭是硬约束。** 工作不能自动占家庭时间。
5. **宁可漏不可错。** 报假承诺比漏掉承诺更致命。
6. **数据是你的。** 可以本地部署，可以随时删除。

---

## Sophia 不做什么

- 不订机票不订酒店——携程/Navan 做预订，Sophia 做智能，互补不竞争
- 不替你做决定——Sophia 建议，你拍板
- 不自动发消息——AI 出错的代价是你的客户信任
- 不做万能聊天助手——不写 PPT、不算税、不闲聊
- 不做团队监控——企业版有聚合仪表盘，但 Sophia 是你的幕僚，不是公司的眼线
