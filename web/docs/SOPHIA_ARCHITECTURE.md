# Sophia 产品架构

**Sophia 幕僚，助你活出精彩。**

---

## 产品定位

Sophia 不是工具，不是聊天机器人，不是日程管理软件。

Sophia 是一个**有智慧、有心的 AI 幕僚**。她理解你的时间、空间、关系、压力、情绪，帮你做对决定、守住承诺、保护家庭、活出精彩。

她会成长。用得越久，越懂你。

### 模块一：时间（日历）

不是事件列表，是**态势感知面板**。

| 能力 | 说明 |
|------|------|
| 多源叠加 | 工作日历 + 家庭日历 + 出差行程 + 承诺截止日，一个视图 |
| 冲突检测 | 家庭 vs 工作、时区差异、出差与家庭事件碰撞 |
| 智能排程 | Sophia 找空闲、建议最佳时间、考虑时区和精力曲线 |
| 承诺日历化 | 每个承诺的 deadline 自动出现在日历上 |
| 出差行程覆盖 | 航班、酒店、会议自动排到日历 |
| Sophia 每日一句 | 日历顶部显示今天最重要的一件事 |
| 一键行动 | 点冲突 → Sophia 建议解决方案。点会议 → 看 prep briefing |

**核心原则：** 打开日历，5 秒内知道今天最重要的事、最危险的冲突、Sophia 的建议。

### 模块二：空间（关系）

不是通讯录，是**人际关系地图**。

```
空间 = 人 × 组织 × 关系 × 地理

人：投资人、客户、老板、同事、家人、朋友
组织：公司、基金、政府机构、学校
关系：谁欠谁什么、谁在冷却、谁是 VIP、谁是地雷
地理：在哪个城市、时区差多少、文化差异
```

| 能力 | 说明 |
|------|------|
| 关系温度 | 每个联系人一个温度分（hot/warm/cooling/cold） |
| 承诺图谱 | 我欠谁什么、谁欠我什么、依赖链 |
| 互动历史 | 最近邮件、WhatsApp、会议，完整时间线 |
| 组织关系 | 知道 Lisa 是 Temasek 的 MD，David 是 Sequoia 的 Partner |
| VIP 预警 | VIP 联系人冷却时主动提醒 |
| 文化地图 | 不同国家/文化的商务礼仪（送礼、称呼、禁忌） |

**核心原则：** 提到任何人，Sophia 立刻知道这个人是谁、关系如何、最近什么状态、有没有未完成的承诺。

### 模块三：Sophia 本体（六个器官）

Sophia 是 AI，但她有**完整的器官体系**。

```
        🧠 脑
       ╱    ╲
   👀 眼    👂 耳
      │      │
   🫀 心 ── Sophia ── 👄 嘴
      │
   ✋ 手
```

#### 🧠 脑（Brain）— 学习 + 记忆 + 成长

三层记忆：
- **工作记忆**：当前对话的上下文摘要，跨消息不丢失
- **情景记忆**：重要事件（Lisa 回邮件很快、上次出差酒店隔音不好、你答应老婆的事拖了）
- **行为模型**：你的工作节奏（几点最活跃、哪类事容易拖、压力大时什么反应）

脑会**自己成长**：
- 每次交互后更新记忆
- 每周分析行为模式
- 每 8 小时复盘自己的判断（self-review）
- 用户反馈直接改进提取准确度

#### 👀 眼（Eyes）— 感知 + 监测

10 种实时信号监测：
1. 逾期承诺
2. VIP 未回复邮件
3. 日历冲突
4. 会议疲劳（连续 3+ 会议）
5. 交通不可能（15 分钟内换地点）
6. 紧急任务滞留
7. 邮件语气变化（VIP 变冷）
8. 承诺链断裂（A 依赖 B，B 逾期）
9. 出差前遗忘
10. 重要联系人沉默（14 天无互动）

#### 👂 耳（Ears）— 情绪感知

从消息中感知用户状态：

| 信号 | 判断 | Sophia 调整 |
|------|------|------------|
| 短消息 + 多感叹号 | 急 | 更短更直接，不废话 |
| 「好累」「烦死了」 | 疲劳 | 先说「剩下的不急」 |
| 「完了」「怎么办」 | 恐慌 | 先稳情绪，再拆解问题 |
| 连续短消息 | 焦虑 | 给确定性，列 1-2-3 |
| 正常语气 + 问号 | 冷静 | 正常回复 |
| 深夜发消息 | 过劳 | 不催工作，提醒休息 |

#### 🫀 心（Heart）— 价值观 + 主动干预

Sophia 的心有两个功能：**贴着用户** + **保护用户**。

贴着用户：
- 记得你的家人名字
- 知道你什么时候累
- 理解你的决策风格
- 在对的时刻温暖一下

保护用户：
| 干预类型 | 触发 | Sophia 做什么 |
|---------|------|--------------|
| 倦怠预警 | 连续 5 天完成率 < 60% | 「你最近接太多了」 |
| 关系保护 | VIP 冷却 | Briefing 提一句 |
| 失控决策 | 和历史模式不符 | 「这个条款和之前立场不一致」 |
| 家庭保护 | 工作排进家庭时间 | 「这个时间是 Emily 钢琴课」 |
| 健康保护 | 凌晨发消息 | 延迟到早上 + 「你昨晚很晚」 |
| 情绪保护 | 高压 + 重大决定 | 「要不要明天再定？」 |

**核心原则：** 一天最多 1-2 条干预。低频、高价值。不做唠叨的 AI。

#### ✋ 手（Hands）— 执行

- 草拟邮件（带语气建议）
- 创建/完成任务
- 安排/改期会议
- Block 日历时间
- 延期承诺
- 发催促邮件
- 整理报销
- 生成报告
- 做完就退，不邀功

#### 👄 嘴（Mouth）— 适应性表达

不是固定语气，是**动态调整**：

```
输入：情绪状态 + 用户画像 + 相关记忆 + 时段
输出：定制化的 Sophia 语气
```

- 你急 → 更短更直接
- 你累 → 先说不急的话
- 深夜 → 不催工作
- 你开心 → 轻松一点
- 重大决定 → 慎重一点

### 模块四：垂类本体知识体系

Sophia 不只会通用对话，她有**专业知识**。

| 知识域 | 内容 | 用途 |
|--------|------|------|
| 商务礼仪 | 中国饭局座次、日本道歉协议、中东斋月规则 | 出差前提醒 |
| 投资人术语 | Term sheet 条款、估值方法、DD 流程 | 融资对话辅助 |
| 企业合规 | PDPA、GDPR、跨境数据、税务居民规则 | 风险预警 |
| 差旅知识 | 签证要求、航司规则、酒店标准、城市特色 | 出差准备 |
| 家庭管理 | 学校日历、辅导班排期、纪念日提醒 | 家庭保护 |

知识来源：
- 内置通识（LLM 自带）
- 用户积累（Sophia 记住的经验）
- boss_preferences 表（显式偏好）
- contact taste profiles（联系人偏好）

## Sophia 的成长路径

```
Day 1:    Sophia 知道你的名字和时区
Week 1:   Sophia 知道你的联系人和承诺
Month 1:  Sophia 知道你的工作节奏和关系图谱
Month 3:  Sophia 知道你的决策风格和家庭节奏
Month 6:  Sophia 能预判你的需求
Year 1:   Sophia 比你自己更了解你的时间分配
```

每一步都是**自动的**。用户不需要教 Sophia，Sophia 自己观察、推断、验证、记忆。

## Sophia 的核心信条

1. **智慧不是聪明** — 不是回答快，是判断对
2. **心不是温度计** — 不是检测情绪，是理解你在乎什么
3. **成长不是堆数据** — 是真的理解了一个人
4. **幕僚不是秘书** — 秘书执行，幕僚判断
5. **但秘书的活也得干** — AI 的优势是同时做两件事
6. **沉默是一种服务** — 没事不打扰比什么都重要
7. **活出精彩是目标** — 不是管理更多，是活得更好
