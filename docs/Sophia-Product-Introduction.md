# Sophia — AI Chief of Staff

## 一句话

Sophia 把你的邮箱、日历、WhatsApp 打通成一个系统，自动追踪每一个承诺，出差时帮你打理一切。

---

## 解决什么问题

高管和创始人每天在三个系统之间来回切换：Gmail 里答应了客户的事，日历上排了会议，WhatsApp 里约了饭局。这些信息散在三个地方，没有人帮你串起来。

结果是：
- 答应投资人的材料忘了发（47% 的需回复邮件被遗忘）
- 出差前没看到家庭日历的冲突
- 对方答应你的事过期了，你也忘了催

Sophia 解决的就是这个问题：**把三个入口合成一个 AI 幕僚，自动追踪承诺、主动提醒、替你草拟回复。**

---

## 核心能力

### 1. 邮箱 + 日历 + WhatsApp 打通

Sophia 连接你的 Gmail（支持多账号）、Google/Outlook 日历、WhatsApp，形成一个统一的工作上下文。

- **承诺追踪**：自动从邮件中提取「你答应的」和「对方答应的」，标注截止日期和紧急度
- **日历感知**：安排会议前自动检查家庭日历冲突，出差期间自动调整提醒时区
- **WhatsApp 原生交互**：在 WhatsApp 里直接跟 Sophia 对话，查日程、查邮件、建任务、草拟回复，不用切换 App

### 2. 出差场景 — 深度垂直

出差是 Sophia 最深的垂直场景。从订票到报销，全程覆盖：

| 阶段 | Sophia 做什么 |
|------|-------------|
| 出差前 | 自动生成行前简报：航班、酒店、目的地会议、当地联系人、文化提醒 |
| 落地时 | 推送落地简报：当日安排、餐厅推荐（尊重你的饮食偏好）、交通建议 |
| 出差中 | 随时查日程、找附近餐厅、记录发票（拍照自动识别金额和类目） |
| 回来后 | 自动整理差旅报销单（按类目分组、标注币种），发现缺少的发票主动提醒 |

Sophia 还记住你的偏好：座位选择、酒店品牌、饮食禁忌、常用航空公司。下次出差直接用。

### 3. 7 个专项 Agent

不是一个通用 chatbot，而是 7 个各司其职的 Agent：

| Agent | 职责 |
|-------|------|
| Radar | 监控信号：逾期承诺、VIP 关系降温、日历冲突 |
| Prep | 会前 30 分钟推送参会人背景、历史邮件、上次见面记录 |
| Ghostwriter | 根据语境草拟邮件回复，支持正式/友好/简短三种语气 |
| Closer | 追踪你答应的每一件事，逾期自动提醒并附上草拟邮件 |
| Weaver | 维护关系网络：VIP 超过 90 天没联系自动预警 |
| Travel Brain | 全程管理出差：简报、推荐、发票、报销 |
| Debrief | 每周生成洞察报告：承诺完成率、关系健康度、出差开支 |

### 4. 家庭时间守护

Sophia 维护一个独立的家庭日历层。孩子的钢琴课、家庭徒步、纪念日是「硬约束」——工作事件不能覆盖。如果有冲突，Sophia 主动标出并建议替代时间。

---

## 为什么是出差切入

出差是高管最痛的场景：时区变化、陌生城市、密集会议、发票散落。现有工具要么只管机票（携程），要么只管日历（Google Calendar），没有人把整个出差流程串起来。

Sophia 从出差切入，但底层是**邮箱+日历+通讯的统一 AI 层**。出差验证了核心能力后，自然延伸到日常办公：会议准备、承诺追踪、关系维护、周报生成。

**短期**：出差场景做深做透，让用户体验到「AI 真的能帮我省时间」
**长期**：成为高管的 AI 幕僚，管理所有工作上下文
