# Chief — 产品需求文档 (PRD)
## 办公协同 AI 助手 | Web App

**版本**: v1.0
**日期**: 2026-03-27
**产品形态**: Web App（浏览器访问，注册即用）
**网址**: https://chief.actuaryhelp.com（待定）

---

## 一、产品一句话

> **Chief 是你的 AI 幕僚长** — 连接你的邮箱、日历和通讯工具，自动告诉你该做什么、回复谁、准备什么。尤其擅长帮出差多的人管好一切。

---

## 二、目标用户

| 画像 | 描述 |
|------|------|
| **谁** | 创始人、高管、销售负责人、咨询顾问 |
| **特征** | 每月出差 2-4 次，跨国跨时区 |
| **痛点** | 邮件太多、会议太密、出差前后混乱、承诺容易忘、报销烦 |
| **现状** | 用 5-8 个 App 拼凑（Gmail + Calendar + WhatsApp + Notes + 报销工具...） |
| **市场** | 新加坡 → 印度 → 美国 → 欧洲 |

---

## 三、用户使用流程

```
用户打开 chief.actuaryhelp.com
    ↓
点击「Connect Your Gmail」→ Google OAuth 授权
    ↓
自动同步邮箱 + 日历（30 秒）
    ↓
进入 Dashboard — 看到今日简报
    ↓
日常使用：查看任务 / 回复邮件 / 看日历 / 跟进事项
    ↓
出差时：出差模式自动激活 → 起飞前必回 / 落地简报 / 出差报告
```

### 模块 10: 智能推荐 — 新加坡（Smart Suggestions - SG）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 会议间隙推荐 | 根据日历空档自动推荐：1 小时空档 → 附近咖啡厅；2 小时空档 → 景点/商场 | P1 |
| 早餐推荐 | 早上第一场会议前，推荐会议地点附近的早餐（kaya toast、本地咖啡店） | P1 |
| 午餐推荐 | 两场会议之间的午餐窗口，按距离+评分推荐，考虑步行/打车时间 | P1 |
| 晚餐推荐 | 最后一场会议结束后，推荐附近晚餐（分档：快餐/小贩中心/商务请客） | P1 |
| 位置感知 | 根据下一场会议的地点，推荐"顺路"的餐厅和去处 | P1 |
| 时间感知 | 精确计算：你有 47 分钟空档 → 推荐步行 5 分钟内的选项 | P1 |
| 本地精选数据 | 新加坡 300+ 精选地点：小贩中心、咖啡厅、商务餐厅、景点，按区域标注 | P1 |
| 推荐卡片 | 日历视图中，空档时段显示推荐卡：名称、评分、步行时间、价位、一句话点评 | P1 |
| Google Maps 跳转 | 点击推荐卡一键打开 Google Maps 导航 | P1 |
| 商务请客模式 | 晚餐推荐中标注"适合请客户吃饭"的餐厅 | P2 |
| 雨天备选 | 检测天气，下雨时推荐室内选项 | P2 |
| 用户收藏 | 收藏喜欢的地点，下次来新加坡自动推荐 | P2 |

**推荐逻辑示例：**
```
日历：
09:00-10:00  会议 @ Raffles Place
              ↓ 空档 1.5 小时
11:30-12:30  会议 @ Marina Bay Sands
              ↓ 午餐窗口 1 小时
13:30-15:00  会议 @ One Raffles Quay

Chief 推荐：
10:00 → "你有 90 分钟。Lau Pa Sat 小贩中心步行 6 分钟，
         推荐沙爹和椰浆饭。或者 The Coffee Bean 步行 3 分钟，
         安静办公。"
12:30 → "午餐 1 小时。CÉ LA VI（MBS 57 楼）适合商务午餐，
         步行 2 分钟，人均 $35-50。或者 Rasapura Masters
         （MBS B2），快餐 $8-15，10 分钟吃完。"
```
